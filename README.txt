== Description ==

Control which HTML tags that can be used in term descriptions. *(Terms are for example post 
tags or post descriptions).*

== Installation ==

1. Extract the ZIP file and move the folder "notes-termcontrol", with it contents, 
   to `/wp-content/plugins/` in your WordPress installation
2. Activate the pluing under 'Plugins' in the WordPress admin area
3. Control your terms under the Notes TermControl menu in the Settings panel
